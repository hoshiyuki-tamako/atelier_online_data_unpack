## Running The Script

### Required

- NodeJs >= 14

```bash
npm i
npm start
```

### Content Update Check List

- copy JSON from AssetStudio to `./export` folder json files
- export Texture2D files to `./docs/img/chara` `./docs/img/enemy` `./docs/img/icon_s`
