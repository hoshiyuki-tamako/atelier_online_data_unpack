## Running The Script

### Required

- NodeJs >= 14

```bash
npm i
```

#### Development Build

```bash
npm start
```

#### Production Build

```bash
npm run build
```

### Content Update Check List

- copy JSON from AssetStudio to `./docs/export/*` folder json files
- export Texture2D files to `./docs/img/*`
- export updated models to `./docs/models/*`
